What the percentage means

Confidence, not certainty.

The percentage reflects source reliability, corroboration across perspectives, primary evidence, claim-review support and missing context. The full evidence view is tucked behind the controls icon on each story. The overview page is built to handle the top 20 when a live feed is connected.

Real News is a static public prototype. It does not scrape full articles, track readers, run comments, or claim absolute truth.
